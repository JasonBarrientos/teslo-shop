# TesloDb API
1. Clonar repo.
2.  Instalar dependencias.
```
npm i
```
3. Clonar el archivo __.env.template__ a uno __.env__.
4. Cambiar las variables de entorno por al correspondientes.
5. Levantar la base de datos.
```
docker-compose up -d
```
6. Levantar modo desarrollo
```
npm run start:dev
```
